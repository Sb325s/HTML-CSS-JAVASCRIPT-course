How to Install and Use VS Code's Live Server Extension

Install Use ideo

Install the Extension

  1. In Visual Studio Code, click the Extensions icon in the left column toolbar.
  2. In the Search bar that appears, type Live Server.
  3. Select the Live Server extension
  4. In the page for Live Server that appears, click on the green install button

Usage Instructions

  • Visual Studio Code must be open to a folder to use Live Server.
  • In Vs Code's Explorer, right click (Windows) or Control+click (macOS) on the HTML file you want to open with Live Server.
